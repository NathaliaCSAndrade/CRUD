# 🧾 Sistema de Controle de Estoque de Adegas (CRUD) em Python
## 📌 Sobre o Projeto:

Este é um sistema de Controle de Estoque de Adegas desenvolvido em Python, executado totalmente via terminal.

O sistema permite realizar operações de CRUD (Create, Read, Update, Delete), utilizando um arquivo `.txt` como base de dados para armazenamento das informações.


## 🚀 Funcionalidades

✔️ Cadastrar novo produto 
✔️ Listar produtos  
✔️ Atualizar lista de produtos  
✔️ Excluir produtos  
✔️ Pesquisar produto  


## 🛠 Tecnologias Utilizadas

- Python
- Manipulação de arquivos (.txt)
- Estrutura modular (separação de funções em arquivo próprio)
